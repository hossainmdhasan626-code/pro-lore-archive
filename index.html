Elowen Ashveil
"The Last Threadweaver"
A mage who can see the fraying edges of the world's magic—and knows that someone is pulling the threads loose on purpose.
Protagonist Mage Aethermoor Third Age
Age 34
Faction The Unravelers
Origin Ashveil Keep
Magic Type Threadweaving
Status Alive
History

Born in the shadow of Ashveil Keep, Elowen showed signs of Threadweaving ability before she could speak. The Aetheric threads that bind the world's magic together were visible to her as faint silver filaments—beautiful, and increasingly frayed. She was taken from her family at age seven by the Conclave of the Woven, trained, and eventually exiled when she refused to stop reporting what she saw: the world's magic was being deliberately unraveled from somewhere deep below the Hollow Peaks.

Motivation

Elowen does not want to save the world out of heroism. She wants to understand it. The fraying she sees is a pattern—elegant, intentional, mathematical. Whoever is unraveling the Aether is doing it like a craftsperson, not a destroyer. She needs to know why before she decides whether to stop them.

Key Events

Survived the Sundering of the Third Gate (Year 312). Discovered the Ashveil Codex beneath her family's ruins. First encounter with Vael Duskborne at the Mirrorfen Crossing. Currently seeks the Oracle Sera in the Unmapped Reaches.

app/api/ — Next.js API Routes (route.ts files)
API Routes

These are the Next.js API routes your frontend components will fetch from. Each is a route.ts file inside the app/api/ folder.

GET /api/characters?world=aethermoor&search=elowen

Returns a list of characters. Supports world and search query params.

Used by: /characters page (SSR) via searchParams

GET /api/characters/[id]

Returns a single character with full lore, relationships, and related events.

Used by: /characters/[id] page (ISR) via generateStaticParams + fetch

GET /api/places?world=aethermoor

Returns all places for a given world.

Used by: /places page (SSG)

GET /api/timeline?world=aethermoor&type=war

Returns timeline events. Filterable by event type. Used by the useTimeline() custom hook.

Used by: /timeline page ('use client' component)

POST /api/characters

Creates a new character entry. Body: { name, worldId, lore, tags }

Used by: "+ New Entry" button → triggers ISR revalidation
